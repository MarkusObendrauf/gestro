goal: make a pie launcher. inspired slightly by CRXMouse chrome extension, but OS-wide.

How it works: when right click is held down, follow the mouse direction. check where it is released. depending on the direction the cursor moved (8 compass directions), send a different shortcut.

Requirements: app must work on linux (KDE), macos, and windows - think of a good tech stack first.

right click action should be invisible (for power users) but there should also be a GUI for configuring the settings - a nice visual interface for every set shortcut where you can change the shortcuts.
